# Налаштування синхронізації нотаток через GitHub Gist

## Що це дає?

- ✅ Нотатки зберігаються в приватному GitHub Gist
- ✅ Автоматична синхронізація кожні 30 секунд
- ✅ Доступ до нотаток з будь-якого пристрою
- ✅ Працює з GitHub Pages (без потреби в сервері)
- ✅ Підтримка всіх браузерів (включно з Firefox)

## Крок 1: Створення GitHub Personal Access Token

1. Перейдіть на https://github.com/settings/tokens/new
2. Заповніть форму:
   - **Note**: `Tab Links Notes Sync` (або будь-яка назва)
   - **Expiration**: `No expiration` (або оберіть термін дії)
   - **Scopes**: Поставте галочку тільки на `gist`
3. Натисніть **Generate token**
4. **ВАЖЛИВО**: Скопіюйте токен (він виглядає як `ghp_xxxxxxxxxxxxxxxxxxxx`)
   - Збережіть його в безпечному місці
   - Після закриття сторінки ви не зможете його побачити знову

## Крок 2: Налаштування в застосунку

1. Відкрийте вкладку **Нотатки**
2. Натисніть кнопку **⚙** (налаштування) в правому верхньому куті
3. У модальному вікні:
   - ✅ Увімкніть чекбокс "Увімкнути синхронізацію з GitHub Gist"
   - Вставте ваш токен у поле "GitHub Personal Access Token"
   - Поле "Gist ID" залиште порожнім (воно заповниться автоматично)
4. Натисніть **"Перевірити з'єднання"** щоб переконатися, що токен працює
5. Натисніть **"Синхронізувати зараз"** щоб створити Gist і завантажити нотатки
6. Натисніть **"Зберегти"**

## Крок 3: Використання на іншому пристрої

Якщо ви хочете отримати доступ до нотаток з іншого комп'ютера/браузера:

1. Знайдіть ID вашого Gist:
   - Перейдіть на https://gist.github.com/
   - Знайдіть Gist з назвою "Tab Links - Notes Data"
   - ID знаходиться в URL: `https://gist.github.com/username/XXXXXXXX` (XXXXXXXX - це ID)
2. На новому пристрої відкрийте налаштування (⚙)
3. Введіть той самий токен
4. Вставте Gist ID
5. Натисніть **"Завантажити з Gist"**
6. Натисніть **"Зберегти"**

## Як це працює?

- **Автоматична синхронізація**: Кожні 30 секунд, якщо є зміни
- **Індикатор статусу**: Кольорова точка біля кнопки налаштувань:
  - 🔵 Синій - синхронізація в процесі
  - 🟢 Зелений - успішно синхронізовано
  - 🔴 Червоний - помилка синхронізації
  - ⚪ Сірий - очікування змін

## Безпека

- ✅ Токен зберігається тільки в localStorage вашого браузера
- ✅ Gist створюється як **приватний** (тільки ви маєте доступ)
- ⚠️ Не діліться токеном з іншими людьми
- ⚠️ Якщо токен скомпрометовано, видаліть його на GitHub і створіть новий

## Відключення синхронізації

1. Відкрийте налаштування (⚙)
2. Зніміть галочку "Увімкнути синхронізацію з GitHub Gist"
3. Натисніть "Зберегти"

Ваші локальні нотатки залишаться в браузері, але синхронізація припиниться.

## Видалення даних

Щоб повністю видалити нотатки з GitHub:

1. Перейдіть на https://gist.github.com/
2. Знайдіть Gist "Tab Links - Notes Data"
3. Натисніть "Delete" в правому верхньому куті

## Усунення проблем

### Помилка "Invalid token"
- Перевірте, що токен скопійовано повністю
- Переконайтеся, що токен має права `gist`
- Можливо, термін дії токена закінчився - створіть новий

### Помилка "Failed to update gist"
- Перевірте інтернет-з'єднання
- Переконайтеся, що Gist не було видалено на GitHub
- Спробуйте створити новий Gist (очистіть поле Gist ID і синхронізуйте знову)

### Нотатки не синхронізуються
- Перевірте, що індикатор показує активність
- Відкрийте консоль браузера (F12) і перевірте наявність помилок
- Спробуйте вимкнути та знову увімкнути синхронізацію
